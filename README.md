## Setup

### Requirements

 - Python 3

```sh
$ pip install -r requirements.txt
```

### Running

You can setup the following env vars:

 - ADDR (default to 'localhost')
 - PORT (default to 3000)
 
 And then:

```sh
$ python server.py
```
