## Setup

### Requirements

 - Python 3
 - Bottle (installed via PIP)

```sh
$ pip install -r requirements.txt
```

### Running

You can setup the following env vars:

 - HOST_ADDR (default to 'localhost')
 - PORT (default to 3000)
 
And then:

```sh
$ python server.py
```

Or even:

```sh
$ PORT=8000 python server.py
```
